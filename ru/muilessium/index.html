Глобальные объекты

/src/muilessium/muilessium.js

Muilessium имеет только один глобальный объект - window.Muilessium.

Он содержит в себе некоторые полезные штуки:

  • EVENTS - Следилка за событиями
  • STORE - Хранилище для данных
  • FACTORY - Основная фабрика компонентов
  • KEYBOARD, MOUSE, TOUCHPAD - Утилиты для устройств ввода
  • UTILS - Немного других утилит
  • DATA_TYPES - Типы данных
  • DEPENDENCIES - Зависимости (для внутреннего использования)
  • POLYFILLS - Полифилы (для внутреннего использования)

Запустите "console.log(Muilessium)" на этой странице, чтобы посмотреть всю структуру.

Небольшой пример ajax-запроса:

const _      = Muilessium.UTILS;
const STORE  = Muilessium.STORE;
const EVENTS = Muilessium.EVENTS;

// Это может быть в одном модуле...
EVENTS.addEvent('data-updated');

// А это в другом...
EVENTS.addEventListener('data-updated', () => {
    console.log(STORE.get('my-data'));
});

// А это - в третьем...
_.ajax.get('https://...', (responseText) => {
    STORE.set('my-data', responseText);
    EVENTS.fireEvent('data-updated');
});

События

/src/muilessium/events.js

У Muilessium есть система глобальных событий.

Основные методы:

  • addEvent(name)
  • addEventListener(name, callback, executeIfAlreadyFired=false)
  • fireEvent(name)

По умолчанию зарегистрированы следующие события:

  • resize-window
  • resize-window-height
  • resize-window-width
  • scroll
  • scroll-start
  • scroll-end
  • app-initialized
  • images-loaded
  • store-updated

Пример использования:

const EVENTS = Muilessium.EVENTS;

EVENTS.addEvent('my-event');

EVENTS.addEventListener('my-event', (state) => {
    console.log(state);
    console.log('it works!');
});

EVENTS.addEventListener('scroll-start', () => {
    EVENTS.fireEvent('my-event');
});

Все компоненты интерфейса имеют аналогичные методы для работы с их собственными событиями.

Хранилище

/src/muilessium/store.js

У Muilessium есть простое глобальное хранилище для данных, которые могут быть нужны разным компонентам.

Основные методы:

  • set(path, data)
  • get(path)

Пример использования:

const STORE = Muilessium.STORE;

STORE.set('my-data.a', 'AAA');
STORE.set('my-data.b', 'BBB');

console.log(STORE.get('my-data.a') + STORE.get('my-data.b'));

Компоненты

/src/muilessium/factory.js
/src/muilessium/base-component.js

У Muilessium есть фабрика компонентов.

Основные методы:

  • registerComponent(type, component)
  • create(type, selector, options)

Все компоненты должны расширять класс BaseComponent.

Пример использования:

const FACTORY = Muilessium.FACTORY;

class MyComponent extends FACTORY.BaseComponent {
    constructor(element, options) {
        super(element, options);
        console.log('ok');
    }
}

FACTORY.registerComponent('MyComponent', MyComponent);
FACTORY.create('MyComponent', 'body'); /* <-- используем body чисто для примера */

console.log(FACTORY.componentsCache.MyComponent);

Стандартный набор компонентов - Muilessium-UI. В примерах ниже будем использовать его.

Если основной dom-элемент компонента имеет id, то инициализированный компонент можно получить используя конструкцию Muilessium.get(componentName, id):

/* Допустим у нас есть аккордеон с id='my-accordion'. */

const accordion = Muilessium.get('Accordion', 'my-accordion');

accordion.unfoldAllItems();

У компонентов могут быть локальные события. Работа с ними аналогична работе с глобальной  системой событий:

// Допустим у нас есть чекбокс.

const checkbox = Muilessium.get('Checkbox', 'my-checkbox');

checkbox.addEventListener('update-state', (state) => {
    console.log(state);
});

У класса BaseComponent есть несколько методов по умолчанию:

  • initAria()
  • initControls()
  • initEvents()
  • addEvent(name)
  • addEventListener(name, callback, executeIfAlreadyFired=false)
  • fireEvent(name)
  • saveState()
  • restoreState()

...и поля:

  • domCache
  • state
  • savedStates

Их следует использовать, чтобы сохранять единообразие в различных компонентах.

Устройства ввода

У Muilessium есть набор утилит для устройств ввода для часто используемых действий:

  • MOUSE
    • onClick(element, callback)
    • onMove(element, callback)
    • onMouseOver(element, callback)
    • onMouseOut(element, callback)
  • KEYBOARD
    • onEnterPressed(element, callback)
    • onSpacePressed(element, callback)
    • onEscapePressed(element, callback)
    • onTabPressed(element, callback)
    • onShiftTabPressed(element, callback)
    • onArrowUpPressed(element, callback)
    • onArrowRightPressed(element, callback)
    • onArrowDownPressed(element, callback)
    • onArrowLeftPressed(element, callback)
  • TOUCHPAD
    • onSwipeLeft(element, callback)
    • onSwipeRight(element, callback)
    • onSwipeUp(element, callback)
    • onSwipeDown(element, callback)
    • onPinchOut(element, callback)

Пример использования:

const MOUSE    = Muilessium.MOUSE;
const KEYBOARD = Muilessium.KEYBOARD;
const TOUCHPAD = Muilessium.TOUCHPAD;

MOUSE.onMove(document, (event) => {
    console.log(event);
});
